# Sistema de Gestión de Acceso - Claro

## Descripción

Este proyecto consiste en el desarrollo de un sistema backend para la gestión de acceso a estaciones base de telecomunicaciones.

Permite controlar el ingreso del personal técnico mediante solicitudes, aprobaciones y registros de acceso.

## Funcionalidades

- Autenticación de usuarios
- Gestión de estaciones base
- Solicitudes de acceso
- Aprobación o rechazo de solicitudes
- Registro de accesos

## Tecnologías utilizadas

- Node.js
- Express
- MySQL

## Instalación

1. Clonar el repositorio
2. Ejecutar el siguiente comando:

npm install

3. Ejecutar el servidor:

node app.js
